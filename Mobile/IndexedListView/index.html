Example of the IndexedListView widget

This widget extends the standard Kendo UI Mobile ListView widget, but adds an index list to the right edge. This was originally written for and demonstrated in the Kendo UI Mobile Music Store sample application. For additional documentation, see: http://docs.kendoui.com/tutorials/Mobile/Kendo%20Mobile%20Music%20Store/kendo-mobile-music-store-indexedlistview-widget
