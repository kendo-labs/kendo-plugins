About Kendo Plugins

Kendo Plugins is a library of third party widgets and controls written for Kendo UI by the community.

Getting Started with Kendo UI Web Plugins

  1. Download the Kendo UI Web library.
    1. Reference the Kendo UI CSS files.
    2. Reference the Kendo UI and jQuery script files.
  2. Download the Kendo UI Web Plugins from https://github.com/kendo-labs/kendo-plugins/tree/master/Web/All
    1. Reference the kendo.web.plugins.min.css file.
    2. Reference the kendo.web.plugins.min.js file.
Example

    <!DOCTYPE html>

    <html>
        <head>
            <meta http-equiv="Content-type" content="text/html; charset=utf-8">
            <link href="//cdn.kendostatic.com/2013.1.319/styles/kendo.common.min.css" rel="stylesheet" />
            <link href="//cdn.kendostatic.com/2013.1.319/styles/kendo.default.min.css" rel="stylesheet" />
            <link href="styles/kendo.web.plugins.css" rel="stylesheet" />
        </head>
        <body>
            <!-- Libs -->
            <script src="//code.jquery.com/jquery-1.9.1.min.js"></script>
            <script src="//cdn.kendostatic.com/2013.1.319/js/kendo.all.min.js"></script>
            <script src="js/kendo.web.plugins.js"></script>
        </body>
    </html>
